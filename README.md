# 🤖 SolanaBot — On-Chain Trading Tool

Automated trading bot built for the Solana blockchain, 
focused on real-time market analysis and pump.fun 
token sniping.

## Features
- ⚡ Real-time price tracking on Solana
- 🎯 Pump.fun new token sniper
- 📊 On-chain buy/sell ratio analysis
- 🔁 Automated entry/exit execution

## Tech Stack
- Node.js / JavaScript
- Helius RPC
- PumpPortal Trade API
- Solana Web3.js
